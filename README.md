# java-cli-maven-spring-failsafe-pmd-mockito-car-data

## Description
A POC for spring app. Testing done with failsafe
and pmd plugins using mockito framework.

## Tech stack
- java
- maven
  - spring
  - failsafe
  - pmd
  - mockito
    - junit5

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- pmd report found at bin/target/site

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
